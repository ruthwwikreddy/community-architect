## Overview

The **Community Architect** portfolio is a high-impact, premium executive showcase designed to highlight expertise in community engineering, software development, and strategic student innovation. Built with a focus on "Tactical Aesthetics," it combines smooth animations with a dark, sophisticated design system.

## Key Features

- **Impact Engine (Events)**: A dedicated system for showcasing global impact milestones and public appearances with interactive multi-angle image galleries.
- **Founder Network**: A curated "Founders & Friends" showcase featuring visionaries and builders with integrated social links and categorized tags.
- **Mission Briefing (Contact)**: A secure, premium communication channel integrated with **Web3Forms** for real-time email delivery.
- **Adaptive Navigation**: Intelligent, responsive navigation that manages cross-page section scrolling and mobile-optimized menus.
- **Legacy Components**: Custom-built sections for **Mission Statement**, **Strategic Tech Stack**, and **Project History**.

## Tech Stack

- **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Forms**: [Web3Forms](https://web3forms.com/)
- **Components**: [Radix UI](https://www.radix-ui.com/) + [shadcn/ui](https://ui.shadcn.com/)

## Project Structure

```text
src/
├── components/         # Premium UI components (Hero, Events, Founders, etc.)
├── data/              # Centralized JSON storage for events and founders
│   ├── events.json    # Legacy & Event archives
│   └── founders.json  # Founder network data
├── pages/             # Main application views (Index, Founders, NotFound)
├── lib/               # Utility functions and theme providers
└── index.css          # Core design tokens and global styles
```

## Setup & Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/ruthwikreddy/community-architect.git
   cd community-architect
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

## Customization

To update the portfolio content without touching the code:

- **Founders**: Modify `src/data/founders.json`
- **Events**: Modify `src/data/events.json`
- **Contact Form**: Update the `access_key` in `src/components/ContactForm.tsx` with your own from [Web3Forms](https://web3forms.com/).
